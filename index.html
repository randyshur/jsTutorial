Functions

Functions allow you to organize and repeat tasks that are used often. They are also useful when it comes to code reuse. Here are two examples of ways to define functions in your code:

  • Older way:
    var myFuncton = function functionName (arguments, commaDelimited) {
    /* The code you want to run goes here */
    }
  • ES6 way (aka fat arrow):
    const myFunction = (arguments, commaDelimited) => { /* Your code goes here */ };

Objects

Objects in JavaScript are simply a way to organize data of any kind and put it into a container. The data is usually organized into name value pairs (ex: dogBreed: "Cattle Dog").

An object may also have methods which are essentialy functions that allow you to retrieve or manipulate information related to the object.

Arrays

Arrays are essentially a list of items. Each item or element on the list can be of any data type including other arrays and even complex objects. Arrays are defined using brackets as opposed to objects which are defined using braces.
let myArray = [element1, element2];

For Loops

For loops allow you to repeat a set of task for a defined set of iterations. The most basic for loops uses the following syntax:
for (let i=0; i < 10; i++) {
/* Your Code to be repeated goes here */
}
The loop above will repeat 10 times. The counter(i) starts at 0, and keeps incrementing by 1(i++) until it equals 10 then it stops.

There is also a for/in loop which will loops through the properties of an object and a for/of loop which was new in ES6 and allows you to iterate through the items in an object.

Conditionals

Conditionals control behavior in JavaScript and determine whether or not pieces of code can run. The most basic is the if statement (if (condition is true {do this})). The if statement can also be followed by one or more else statements that will be executed if the initial if fails. There is also a switch statement which is just a more concise way of writing multiple if-else-if-elses.
